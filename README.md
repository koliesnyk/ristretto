# Ristretto

A dark theme for [Zed](https://zed.dev) with a muted, low-contrast palette inspired by One Dark — easy on the eyes for long coding sessions.

## Installation

### Manual (Dev Extension)

1. Clone the repository:
   ```bash
   git clone https://github.com/koliesnyk/ristretto.git
   ```
2. In Zed open **Extensions** (`Cmd+Shift+X`) → **Install Dev Extension** → select the cloned folder
3. Add to your `settings.json`:
   ```json
   "theme": "Ristretto",
   "icon_theme": "Ristretto"
   ```
